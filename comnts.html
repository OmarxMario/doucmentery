📦 UI Framework Documentation


🎨 Theme Optimization

The framework includes 7 built-in primary colors: red, blue, green, yellow, orange, purple, and pink. To apply a global theme, add the color class to the body tag:

<body class="[your-color]"></body>

We also provide 5 variations mapped directly to your active theme (.lgst for lightest, .lg for light, .ac for accent, .dk for dark, and .dkst for darkest). Apply them using the background or text utility prefixes:

<div class="['bg-[variant]' for backgrounds, 'tx-[variant]' for text]"></div>

📐 Proportions

Manage dimensions dynamically using our responsive custom properties instead of hardcoded CSS values. For external layout resources, you can use the reference link below.

To explicitly define element width:

<div class="w" style="--w: [value];"></div>

To explicitly define element height:

<div class="h" style="--h: [value];"></div>
